В работе исследована задача централизованного управления группой взаимосвязанных систем в реальном времени. 
Для исследуемой задачи сформулированы подходы ее сведения к задаче линейного программирования, предложен алгоритм работы централизованного оптимального регулятора. В дальнейшем с помощью последовательного подхода построена функциональная форма задачи, а алгоритм оптимального регулятора реализован в MATLAB с применением на втором шаге стандартной процедуры linprog. Проведен анализ программного решения и реализации обратной связи.


Дальнейшее развитие работы связанно с управлением группой в децентрализованном случае, то есть в случае, где каждый объект управления имеет собственный регулятор.
